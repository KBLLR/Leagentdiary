# Experimental Stage Quarantine

This directory preserves stage, scene, and deployment-timeline experiments that were previously mixed into the active LeAgentDiary house.

## Status
- preserved for reference
- explicitly non-blocking
- not part of the default runtime
- not required for `npm run build`
- not required for `npm run export:traces`
- historical HTML entrypoints have been renamed with `.experimental.snapshot`

## What lives here
- former `src/stage/` code
- former `public/stage/` and related stage-only assets
- former `api/vercel-deployments.mjs`
- former `server/fetchVercelDeployments.mjs`
- historical stage planning docs

## Boundary
LeAgentDiary’s active product lane is session chronology review plus curated reflection export for Anthology.

Anything in this directory is secondary experimental material and must not redefine the house as a scene platform, memory backend, or stage service.
